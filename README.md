# 【精密解析】そらる・シンクロ率チェッカー

あなたの歌声が、歌い手「そらる」さんにどれだけ近いかを音声解析で判定するアプリです。

## 使い方

1. 自分の歌声（wav / mp3）をアップロード
2. 「精密解析スタート」を押す
3. シンクロ率と、あなたに近いそらる楽曲 TOP5 が表示されます

## 推奨する音声

- 長さ：10〜20秒
- 内容：サビや盛り上がり部分（声が大きいところ）
- アカペラ・声だけだとより精度が上がります

## 技術仕様

- 音声特徴量：MFCC（13次元 × mean/std = 26次元）
- 解析ライブラリ：librosa
- 比較手法：ユークリッド距離
- 参照データ：そらるさんの楽曲 28曲
